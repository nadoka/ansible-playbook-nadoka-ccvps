# ansible-playbook-nadoka-ccvps

Playbook for nadoka-ccvps.n-z.jp

## Usage

### Ansible in Guest OS

```
    % git clone https://github.com/nadoka/ansible-playbook-nadoka-ccvps
    % cd ansible-playbook-nadoka-ccvps
    % vagrant up
```

### Ansible in Production

```
    $ git clone https://github.com/nadoka/ansible-playbook-nadoka-ccvps
    $ cd ansible-playbook-nadoka-ccvps
    $ ./provision/provision.sh
```
